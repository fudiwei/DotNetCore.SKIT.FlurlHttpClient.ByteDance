## 简体中文 / Simplified Chinese

基于 `Flurl.Http` 的 TikTok Shop Open Platform HTTP API SDK。

### 【功能特性】

-   基于 TikTok Shop Open Platform API 封装。
-   请求时自动生成签名，无需开发者手动干预。
-   提供了 HMAC-SHA-256 等算法工具类。
